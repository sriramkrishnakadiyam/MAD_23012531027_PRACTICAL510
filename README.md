AIM:

Create an Android application containing Login and Registration screens built entirely using Jetpack Compose.

This application demonstrates how to build UI screens using Jetpack Compose and navigate between them using the Navigation component.

Features
1. Login Screen

A simple login UI

Contains Email and Password fields

Includes a button to navigate to the Registration screen

2. Registration Screen

Contains fields for:

Name

Phone Number

City

Email

Password

3. Jetpack Compose UI

Entire user interface built using Jetpack Compose

Demonstrates modern, declarative UI development

4. Navigation Component

Handles navigation between:

Login Screen

Registration Screen

Concepts Used
1. Jetpack Compose

A modern UI toolkit that:

Uses a declarative approach

Reduces boilerplate code

Offers powerful and intuitive Kotlin-based APIs

Helps build fast, responsive, and modern Android UI

2. Jetpack Navigation

A component that:

Manages navigation between screens

Supports simple and complex navigation flows

Integrates smoothly with Compose

3. State Management

Uses remember and mutableStateOf

Ensures the UI updates automatically when data changes

Helps manage input fields and button states

4. Kotlin

Entire app written in Kotlin

Offers concise, safe, and expressive syntax

Works seamlessly with Jetpack Compose

5. Material Design

UI components follow Material Design guidelines

Provides a clean, modern, and consistent look

Uses Material theming, buttons, text fields, etc
